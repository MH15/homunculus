# homunculus

Homunculus is an agentic assistant (think Claude Code, Codex, etc) written in [Effect](https://effect-ts.github.io).

## Development

To install dependencies:

```bash
bun install
nvm use 24
```

To run:

```bash
node --env-file=.env index.ts
```
